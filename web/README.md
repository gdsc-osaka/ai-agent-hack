# Web Frontend

## Getting Started
```bash
# Install dependencies
pnpm install

# Start the development server
npm run dev

# Run the linter
npm run lint
# Fix linting issues and format code
npm run fix

# Build the project
npm run build
```

## Directory Structure
```
/
└─ src/
   ├─ app/           // App Router (https://nextjs.org/docs/app)
   │  ├─ some-page/  // 他のページ
   │  │  ├─ page.tsx
   │  │  └─ ...
   │  ├─ atom.ts     // jotai (https://jotai.org/)
   │  └─ page.tsx
   ├─ components/
   │  ├─ ui/         // Generated by shadcn/ui (https://ui.shadcn.com/)
   │  └─ *.tsx       // ドメイン寄りのコンポーネント
   ├─ lib/           // library
   └─ openapi/       // Generated by .scripts/gen_openapi.sh
```
